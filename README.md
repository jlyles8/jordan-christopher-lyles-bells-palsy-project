# Bell’s Palsy Eyelid Prosthesis Proof of Concept

Portfolio website for an undergraduate biomedical engineering design project by Jordan Christopher Lyles and teammates.

## GitHub Pages upload
1. Create a new GitHub repository.
2. Upload `index.html`, `style.css`, and the full `assets/` folder.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/root`, then save.

The included PDF and page images are stored in `assets/` so the page works as a complete static site.
